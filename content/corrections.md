# Corrections Policy

The Donor Map publishes corrections on any material factual error. Every claim in our database traces to a cited source through our source registry. If a source is wrong, we fix it.

## How to report an error

**Email:** guerillapropaganda@proton.me

**GitHub:** [File a correction request](https://github.com/Guerillapropaganda/donor-map-site/issues/new?template=correction-request.yml)

Include:
- Which profile or page contains the error
- What the error is (be specific)
- What the correct information is, with a source if possible

## Our process

1. We acknowledge receipt within 48 hours
2. We investigate the claim against our source registry
3. If confirmed, we publish a correction within 7 days
4. All corrections are logged publicly below

We do not remove factual information that is accurately sourced. We correct errors.
